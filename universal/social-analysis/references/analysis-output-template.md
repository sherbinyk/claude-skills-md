# Analysis Output Template

Use this template whenever writing the written analysis that accompanies the unified Excel file.
The analysis is written in the conversation — NOT inside the Excel file itself.

---

## Language & Client Defaults

| Client | Language | Arabic Register | Platforms |
|---|---|---|---|
| Link Development | English | — | Facebook, Instagram, LinkedIn |
| Flamingo | Arabic | Egyptian (مصري) | Facebook, Instagram, TikTok |
| Qormuz | Arabic | Gulf / KSA (خليجي) | Facebook, Instagram, Snapchat |
| Other / unknown | Match user's language or ask | — | Whatever was uploaded |

**If writing in Arabic:**
- Use natural marketing Arabic — not formal MSA
- Egyptian register: casual, warm, direct (e.g. "إيه اللي شغّل أكتر؟")
- Gulf/KSA register: slightly more formal but still conversational (e.g. "ما الذي حقق أفضل أداء؟")
- All section headers and metric labels in Arabic
- Numbers in Arabic-Indic numerals are acceptable but Latin numerals (1,234) are fine too
- Do NOT mix English and Arabic in the same analysis unless the user explicitly asks

---

## When to write which sections

| Situation | Sections to include |
|---|---|
| Single platform only | Overall Summary + Content Performance + Top Posts + Recommendations |
| 2–3 platforms | All sections |
| First-time / onboarding report | All sections + a "Data Notes" section flagging gaps |
| Quick check ("how did last week go?") | Overall Summary + Top Posts + 1–2 recommendations |

### Recommendations

3–5 specific, actionable recommendations. Each one should state what to do AND why.
Write for a decision-maker — not a data analyst.

**Format:**
> **[Action verb] [specific thing]** — [reason, based on data from this report].

**Examples:**
> **Increase video production on Facebook** — video posts generated 2× the ER of image posts
> this quarter. Even one extra video per month could meaningfully move the platform average.

> **Use carousels more on Instagram** — carousel posts averaged 2.8% ER vs 0.9% for images.
> They likely benefit from the swipe interaction driving higher algorithmic reach.

> **Test LinkedIn video** — the 6 video posts averaged 4.7% ER this quarter vs 1.3% page average.
> Video is LinkedIn's highest-performing format and currently underused.

> **Post cross-platform content selectively** — some posts perform very differently per platform
> (e.g., the Mother's Day video: 5.1% on FB, 0.6% on IG). Not all content is right for all channels.

> - 4 Facebook posts have no Instagram equivalent and were excluded from cross-platform comparison.

## Language and Tone Guidelines

- **Language:** Match the client default (see table above). If the user writes in a different
  language from the client default, follow the user.
- Lead with outcomes, not numbers. "LinkedIn drove the highest engagement this quarter" is
  better than "LinkedIn had an average ER of 1.3%."
- Do not explain what metrics mean (impressions, ER) unless the user seems unfamiliar.
- Keep the full analysis under 700 words in conversation — the details are in the Excel file.
- Avoid hedging language ("it seems like", "possibly", "might"). Be direct.
- For Arabic: Gulf register for Qormuz (KSA). Egyptian register for Flamingo.
- TikTok-specific: mention watch time and completion rate when available — these are the
  primary performance signals on TikTok and have no equivalent on other platforms.
- Snapchat-specific: note that ER excludes likes (Snapchat has none) — Swipe-Ups are the
  most meaningful engagement signal on Snapchat.

### Data Completeness Score (ALWAYS include — last section of every analysis)

This section is mandatory. Always end the analysis with it, even if the score is 100%.

**Format:**

```
---
**Data Completeness: [score]% — [High Confidence / Moderate Confidence / Low Confidence]**

Issues identified:
- [Issue 1]
- [Issue 2]

Notes:
- [Note 1]
```

**Score thresholds:**
- 85–100%: High Confidence
- 70–84%: Moderate Confidence — analysis is valid but has noted gaps
- Below 70%: Low Confidence — recommend re-uploading missing files

**Tone for this section:** Factual, neutral. Do not editorialize. List issues exactly as computed.
This is not a judgement on content performance — it reflects data completeness only.

**Example output:**

```
---
**Data Completeness: 82% — Moderate Confidence**

Issues identified:
- Instagram does not export click data in native analytics — Clicks column is blank for all IG posts. (-5%)
- LinkedIn Content Type is blank for 32 of 38 posts (84%) — labeled as 'Image/Text'. Content format cannot be confirmed for these. (-3%)
- Instagram follows export has 57 days recorded vs Facebook's 87. IG only logs days with ≥1 new follower — days with zero are missing. (-2%)

Notes:
- LinkedIn post-level reach is not available in the native export. Use the LinkedIn Daily Metrics sheet for page-level reach trends.
```
