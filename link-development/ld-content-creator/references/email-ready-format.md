# Email-Ready Content Format

Use this format when the user asks for an "email ready" version, "email table", or "Outlook format" of any content.

This format is designed to be pasted directly into Outlook Desktop as a structured briefing table — sent to design teams, social media schedulers, or stakeholders.

---

## Table Structure

Produce a table with exactly these 6 columns in this order:

| Column | Description | Rules |
|---|---|---|
| **Milestone** | Sequential number for the content batch | Start at 1, increment per post |
| **Product** | Product or service name being promoted | Use the exact product name (e.g., Digital Courts, Strategy360) |
| **TOV** | Text on Visual — the creative headline | Short, punchy, 1 line. Same as the TOV already written |
| **Creative CTA** | CTA text to appear on the creative | Write the CTA if one is needed; write "No need" if the post is awareness/thought leadership |
| **Logos** | Logo placement instruction | Always format as: "Link Development > [Product Name]" |
| **Caption** | The full post copy | Full caption text, no hashtags in this column — clean paragraphs only |

---

## Output Format

Always deliver as a Markdown table so it pastes cleanly into Outlook:

```
| Milestone | Product | TOV | Creative CTA | Logos | Caption |
|---|---|---|---|---|---|
| 1 | [Product] | [TOV text] | [CTA or "No need"] | Link Development > [Product] | [Full caption text] |
| 2 | ... | ... | ... | ... | ... |
```

---

## Rules

- **One row per post** — if the user has 5 posts, the table has 5 rows
- **Caption column**: paste the full post copy as plain text — no bold, no bullet points, no hashtags
- **Milestone numbers**: always start at 1 and go up sequentially unless the user specifies otherwise
- **TOV**: pull directly from the TOV already written for each post — do not rewrite
- **Logos**: always "Link Development > [Product Name]" unless the user specifies a different logo combination
- **Creative CTA**: use "No need" for Thought Leadership, Awareness, and Engaging post types — use an actual CTA for Promotional and Informative posts

---

## When to Use

Trigger this format when the user says any of the following:
- "email ready"
- "email version"
- "Outlook table"
- "send to the team"
- "briefing table"
- "ready to send"

Apply it to whatever content was just produced in the conversation — no need to rewrite the content, just restructure it into the table.
