# SEO Tool Column Name Mappings

Use this file to rename exported columns into the standard internal schema before analysis.

---

## Google Search Console

| GSC Column | Standard Name |
|---|---|
| Top queries / Query | `keyword` |
| Clicks | `clicks` |
| Impressions | `impressions` |
| CTR | `ctr` |
| Position | `avg_position` |
| Top pages / Page / Landing Page | `url` |
